```

- Hover: shadow-md + slight scale

- Default: solid color
- Hover: darker shade + shadow

- Default: closed
- Hover: highlight
- Active: open with options

### Click States
```
Search Input:
- Focus: blue ring
- Active: cursor visible

Filters:
- Click: dropdown opens
- Select: updates list

- Click: download action

- Essential info visible

### Mobile (< 768px)
- 1-column stats grid
- Minimal student cards
- Vertical filters
- Priority info only
